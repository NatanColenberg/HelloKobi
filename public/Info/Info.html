Docker POC Project

Project Info:

This project is a POC for Docker Containers inner communication.
All containers are hosted in GCP (Google Cloud Platform) using the GKE (Google Kubernetes Engine).
The first container is the Web Server, it's a simple NodeJs server using Express.
This Web Server is serving all client-side reasources for the app as well as fetching data from the second docker container.
The second container is the Stock Price Server, also using NodeJs to recieve the stock symbol from a client
and return the price of that stock from Yahoo Finence API (in our example stock symbol = 'CHNG' - Change Healthcare).
The Web Server communicates with the Stock Price Server using Local DNS to expose it's service.
The Web Server sits behinde a Load Balancer that exposes an external IP address to communicate with that container
while the Stock Price Server is configured as a Cluster IP so that it does not exposes any external IP address
and can only communicate with other containers in it's cluster using his internal IP or the local DNS name that is provided by it's service.
